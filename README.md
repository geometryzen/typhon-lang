# typhon-lang

Python Tools for Scanning, Parsing, Semantic Analysis, and Code Generation.

Based on the SPARK tool chain:

http://pages.cpsc.ucalgary.ca/~aycock/spark/

## Installation

```
npm install
npm update
```

## Build

```
npm run build
```

## Text

```
npm run test
```

## Development

```
npm run watch
```
